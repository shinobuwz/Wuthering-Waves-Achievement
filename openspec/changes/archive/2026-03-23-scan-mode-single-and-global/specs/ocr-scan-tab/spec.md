## 修改需求

### 需求:扫描控制
Tab 页必须提供开始和停止扫描的控制，并根据当前选中的扫描模式执行不同的扫描逻辑。

#### 场景:开始扫描
- **当** 游戏窗口已检测到，用户点击"开始扫描"按钮
- **那么** 系统必须根据当前选中的扫描模式启动对应的 OCR 扫描流程：单页扫描模式调用单页扫描逻辑，全局扫描模式调用全Tab遍历逻辑；按钮变为"停止扫描"

#### 场景:停止扫描
- **当** 扫描正在进行中，用户点击"停止扫描"按钮
- **那么** 系统必须在当前轮次完成后安全停止扫描

#### 场景:未检测窗口时点击扫描
- **当** 游戏窗口未检测到，用户点击"开始扫描"
- **那么** 系统必须提示用户先检测游戏窗口

### 需求:实时进度显示
Tab 页必须在扫描过程中实时显示进度信息，格式根据扫描模式区分。

#### 场景:显示扫描进度
- **当** 扫描正在进行中
- **那么** 全局扫描模式必须显示：当前一级/二级分类名、已识别成就数量、已完成数量；单页扫描模式必须显示："单页扫描中"、已识别成就数量、已完成数量
