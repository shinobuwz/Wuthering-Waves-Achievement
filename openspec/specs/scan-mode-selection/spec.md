# scan-mode-selection

## 目的

在 OCR 扫描控制区域提供扫描模式选择控件，允许用户在「单页扫描」和「全局扫描」之间切换。

## 需求

### 需求:扫描模式选择控件
系统必须在扫描控制区域提供扫描模式选择控件，允许用户在「单页扫描」和「全局扫描」之间切换。

#### 场景:默认模式
- **当** OCR 扫描 Tab 页面加载完成
- **那么** 扫描模式必须默认选中「全局扫描」

#### 场景:切换模式
- **当** 用户通过下拉框选择不同的扫描模式
- **那么** 系统必须记录当前选中的模式，后续点击"开始扫描"时按选中模式执行

#### 场景:扫描中禁止切换
- **当** 扫描正在进行中
- **那么** 系统必须禁用模式选择控件，防止用户在扫描过程中切换模式

#### 场景:扫描结束恢复
- **当** 扫描完成或被用户停止
- **那么** 系统必须重新启用模式选择控件
